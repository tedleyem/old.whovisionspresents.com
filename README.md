### Setup

Get the code

<pre>git clone https://github.com/ubaimutl/whovisionspresents.git</pre>
 
Install required dependencies

<pre>npm run install </pre>


Start the server

<pre>npm run start</pre>

### More

Modify pages content in  `src/content_option.js`.
